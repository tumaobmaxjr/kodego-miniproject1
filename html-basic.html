HTML Basic Examples

In this chapter, we'll go over some fundamental HTML examples.

Don't worry if we use tags you haven't seen before.

HTML Documents

All HTML documents must start with a document type declaration: <!DOCTYPE html>.

The HTML document itself begins with <html> and ends with </html>.

The visible part of the HTML document is between <body> and </body>.

<!DOCTYPE html>
<html>
<body>

<h1>My first heading</h1>

<p>My first paragraph.</p>

</body>
</html>

The <!DOCTYPE> Declaration

The <!The DOCTYPE> declaration represents the document type and assists browsers in correctly displaying web pages.

It must appear only once, at the top of the page (before any HTML tags).

The <!The DOCTYPE> declaration is case insensitive.

The HTML5's <!DOCTYPE> declaration is:

<!DOCTYPE html>

HTML Headings

HTML headings are defined by the <h1> to <h6> tags.

The most important heading is defined by <h1>.The least important heading is defined by <h6>:

<h1>This is heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>

HTML Paragraphs

The <p> tag in HTML defines paragraphs:

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

HTML Links

The <a> tag is used to define HTML links:

<a href="https://www.codestarter.com">This is link</a>

The href attribute specifies the link's destination.

Attributes are used to add extra information to HTML elements.

In a later chapter, you will learn more about attributes.

HTML Images

The <img> tag in HTML is used to define images.

As attributes, the source file (src), alternative text (alt), width, and height are provided:

<img src="codestarter.png" alt="CodeStarter.com" width="100" height="250">

How to View HTML Source?

Have you ever seen a Web page and thought to yourself, "Hey! How did they do that?"

See the HTML Source Code

Right-click an HTML page and choose "View Page Source" (in Chrome) or "View Source" (in Edge), or something similar in other browsers. This will open a window containing the page's HTML source code.

Inspect an HTML Element:

Right-click on an element (or a blank area), and choose "Inspect" or "Inspect Element" to see what elements are made up of (you will see both the HTML and the CSS) (you will see both the HTML and the CSS). You can also make live changes to the HTML or CSS in the Elements or Styles panels that appear.
