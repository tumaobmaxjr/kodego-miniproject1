HTML Attributes

HTML attributes offer further details about HTML elements.

HTML Attributes

  • Every element in HTML can have attributes.
  • Additional information about elements is provided by attributes.
  • Always specify attributes in the start tag.
  • Typically, attributes are expressed as name="value".

The href Attribute

A hyperlink is defined using the <a> tag. The URL of the page to which the link points is specified by the href attribute:

<a href="https://www.codestarter.com">CodeStarter</a>

The src Attribute

To embed an image in an HTML page, use the <img> tag. The src attribute specifies the path to the displayed image:

<img src="sample_img.jpg">

The URL can be specified in the src property in one of two ways:

1. Absolute URL - links to an image hosted on another domain that is external.

Example: src="https://www.codestarter.com/images/sample_img.jpg".

Note: Copyright may apply to external photos. You might be breaking copyright laws if you use it without authorization. Additionally, you have no control over external photos; they could be changed or withdrawn at any time.

2. Relative URL - references to an image hosted on the internet. Here, the domain name is absent from the URL. The URL will be relative to the current page if it starts out without a slash.

Example: src="sample_img.jpg"

The URL will be relative to the domain if it starts with a slash.

Example: src="/images/sample_img.jpg".

Tip: Relative URLs are usually always the best choice. If you switch domains, they won't stop working.

The width and height Attributes

The width and height attributes, which specify the width and height of the image (in pixels), should also be included in the <img> tag:

<img src="sample_img.jpg" width="250" height="500">

The alt Attribute

The required alt attribute for the <img> tag specifies alternate text for an image if it cannot be displayed for some reason. This could be because of a slow connection, an error in the src attribute, or the user using a screen reader.

<img src="sample_img.jpg" alt="Sample Image">

The style Attribute

Color, font, size, and other styles can be added to an element using the style attribute.

<p style="color: red;"> This is a red paragraph.</p>

The lang Attribute

To declare the language of the Web page, always include the lang attribute inside the <html> tag. This is intended to help search engines and browsers.

The language in the following example is English:

<!DOCTYPE html>
<html lang="en">
<body>
...
</body>
</html>

In the lang attribute, country codes can be added to the language code. As a result, the first two characters of the HTML page define the language, and the last two characters define the country.

The example below uses English as the language and the United States as the country:

<!DOCTYPE html>
<html lang="en-US">
<body>
...
</body>
</html>

The title Attribute

The title attribute provides additional information about an element.

When you hover your mouse over the element, the value of the title attribute will be displayed as a tooltip:

<p title="I'm a tooltip"> This is a paragraph.</p>

We recommend that you always use lowercase attributes.

Lowercase attribute names are not required by the HTML standard.

The title attribute (as well as all other attributes) can be written in uppercase or lowercase, such as title or TITLE.

We recommend that you always quote attribute values.

Quotes around attribute values are not required by the HTML standard.

Good:
<a href="https://www.codestarter.com">CodeStarter</a>
Bad:
<a href=https://www.codestarter.com>CodeStarter</a>

Chapter Summary

  • Attributes can be applied to any HTML element.
  • The hrf attribute of <a> specifies the URL of the page to which the link leads.
  • The path to the image to be displayed is specified by the src attribute of <img>.
  • The width and height attributes of <img> provide image size information.
  • The alt attribute of <img> specifies an image's alternate text.
  • The style attribute is used to apply styles to an element, such as color, font, size, and other options.
  • The <html> tag's lang attribute specifies the language of the Web page.
  • The title attribute provides additional information about an element.
