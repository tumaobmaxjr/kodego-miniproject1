CSS Selectors

A CSS selector is used to choose which HTML elements to style.

CSS selectors are used to "find" (or choose) the HTML elements to style.

CSS selectors are classified into five types:

  • Selectors that are simple (select elements based on name, id, class)
  • Selectors for combinations (select elements based on a specific relationship between them)
  • Selectors for pseudo-classes (select elements based on a certain state)
  • Selectors for pseudo-elements (select and style a part of an element)
  • Selectors for attributes (select elements based on an attribute or attribute value)

This page will go over the most fundamental CSS selectors.

CSS element Selector

The element selector chooses HTML elements based on their names.

Example

All <p> elements on the page will be center-aligned and have red text:

p {
    text-align: center;
    color: red;
}

CSS id Selector

The id selector uses an HTML element's id attribute to select a specific element.

Because an element's id is unique within a page, the id selector is used to select only one element!

To select an element with a specific id, use the hash (#) character followed by the element's id.

Example

The following CSS rule will be applied to the HTML element with id="par1":

#par1 {
    text-align: center;
    color: red;
}
Note: An id name cannot begin with a number!

CSS class Selector

The class selector is used to select HTML elements that have a specific class attribute.

To select elements of a particular class, use a period (.) character followed by the class name.

Example

All HTML elements with class="center" will be red and center-aligned in this example:

.center {
    text-align: center;
    color: red;
}

A class can also be configured to affect only specific HTML elements.

Example

Only in this case, elements with the class="center" will be highlighted in red and center-aligned:

p.center {
    text-align: center;
    color: red;
}

HTML elements can also refer to multiple classes.

Example

In this case, the following elements will be styled with class="center" and class="large":

<p class="center large">This paragraph refers to two classes.</p>
Note: A class name cannot begin with a number!

CSS Universal Selector

All HTML elements on the page are selected by the universal selector (*).

Example

Every HTML element on the page will be affected by the CSS rule below:

* {
    text-align: center;
    color: blue;
}

CSS Grouping Selector

All HTML elements with the same style definitions are selected by the grouping selector.

Examine the CSS code below (the h1, h2, and p elements all have the same style definitions):

h1 {
    text-align: center;
    color: red;
}

h2 {
    text-align: center;
    color: red;
}

p {
    text-align: center;
    color: red;
}

To reduce code, it is preferable to group the selectors.

To group selectors, use a comma to separate each one.

Example

In this example, we've grouped the selectors from the previous code:

h1, h2, p {
    text-align: center;
    color: red;
}

Selector Example Example description
#id #firstName Selects the element with id="firstName"
.class .intro Selects all elements with class="intro"
element.class p.intro Selects only <p> elements with class="intro"
* * Selects all elements
element p Selects all <p> elements
