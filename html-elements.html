HTML Elements

A start tag, some content, and an end tag define an HTML element.

Everything from the start tag to the end tag is an HTML element:

<tagname>Content Goes here...</tagname>

Some HTML elements are as follows:

<h1>My first heading.</h1>

<p>My first paragraph.</p>

Start tag Element content End tag
<h1> My first heading </h1>
<p> My first paragraph </p>
<br> none none

Note: Some HTML elements, such as the <br> element, have no content. These are known as empty elements. There is no end tag for empty elements!

Nested HTML Elements

HTML elements are capable of being nested (this means that elements can contain other elements).

Every HTML document is made up of nested HTML elements.

The example below contains four HTML elements (<html>, <body>, <h1>, and <p>):

<!DOCTYPE html>
<html>
<body>

<h1>My first heading</h1>
<p>My first paragraph.</p>

</body>
</html>
Example Explained

The root element is the <html> element, which defines the entire HTML document.

It has an <html> start tag and an </html> end tag.

Then there is a <body> element inside the <html> element:

<body>

<h1>My first heading</h1>
<p>My first paragraph.</p>

</body>

The <body> element defines the body of the document.

It has a <body> start tag and a </body> end tag.

Then, within the <body> element, there are two additional elements: <h1> and <p>:

<h1>My first heading</h1>
<p>My first paragraph.</p>

A heading is defined by the <h1> element.

It has a <h1> start tag and a </h1> end tag:

<h1>My first heading</h1>

A paragraph is defined by the <p> element.

It has a <p> start tag and a </p> end tag:

<p>My first paragraph.</p>

Never, Ever Skip the End Tag

Some HTML elements will display correctly even if the end tag is omitted:

<html>
<body>

<p>This is a paragraph.</p>
<p>This is a paragraph.</p>

</body>
</html>
Note: However, you should never rely on this! If you forget the end tag, you may get unexpected results and errors!

HTML Elements With No Content

HTML elements that have no content are referred to as empty elements.

The <br> tag defines a line break and is an empty element that does not have a closing tag:

<p>This is a <br> paragraph with a line break.</p>

HTML is Not Case Sensitive

HTML tags are not case sensitive: <P> and <p> have the same meaning.

Although the HTML standard does not require lowercase tags, the W3C recommends and requires lowercase in stricter document types such as XHTML.
