How to Include CSS

A browser will format an HTML document in accordance with the information in a style sheet when it reads one.

Three CSS Insertion Methods:

  • External CSS
  • Internal CSS
  • Inline CSS

External CSS

By modifying only one file in an external style sheet, you can alter the appearance of a complete website!

Each HTML page must provide a link to the external style sheet file in the head section's <link> element.

Example

The <link> element, used in the <head> section of an HTML page, defines external styles:

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="style.css">
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

An external style sheet can be created in any text editor and saved with the extension.css.

There should be no HTML tags in the external.css file.

The "style.css" file looks like this:

"style.css"
body {
    background-color: linen;
}

h1 {
    color: maroon;
    margin-left: 20px;
}
Note: Please keep in mind that there should be no space between the property value and the unit:
Incorrect (space): margin-left: 20 px;
Correct (nospace): margin-left: 20px;

Internal CSS

If a single HTML page has a distinct style, an internal style sheet may be used.

The internal style is defined inside the <style> element, inside the head section.

Example

Internal styles are defined within the <style> element of an HTML page's <head> section:

<!DOCTYPE html>
<html>
<head>
<style>
body {
    background-color: linen;
}

h1 {
    color: maroon;
    margin-left: 20px;
}
</style>
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

Inline CSS

An inline style can be used to apply a distinct look to a single element.

Add the style attribute to the relevant element to use inline styles. Any CSS property can be included in the style attribute.

Example

Inline styles are defined within the relevant element's "style" attribute:

<!DOCTYPE html>
<html>
<head>

<h1 style="color: blue; text-align: center;">This is a heading</h1>
<p style="color: red;">This is a paragraph.</p>

</body>
</html>
Tip: Inline styles lack many of the benefits of style sheets (by mixing content with presentation). Use this method with caution.

Multiple Style Sheets

If some properties have been defined for the same selector (element) in different style sheets, the value from the last read style sheet will be used.

Assume that an external style sheet has the following <h1> element style:

h1 {
    color: navy;
}

Then, assume that an internal style sheet also has the following style for the element:

h1 {
    color: orange;
}
Example

If the internal style is defined after the link to the external style sheet, the following elements will be "orange":

<head>
<link rel="stylesheet" type="text/css" href="style.css">
<style>
h1 {
    color: orange;
}
</style>
</head>
Example

If the internal style is defined before the external style sheet link, the following elements will be "navy":

<head>
<style>
h1 {
    color: orange;
}
</style>
<link rel="stylesheet" type="text/css" href="style.css">
</head>

Cascading Order

When more than one style is specified for an HTML element, which one is used?

The following rules will "cascade" all the styles on a page into a new "virtual" style sheet, with number one having the highest priority:

  1. Inline style (inside an HTML element)
  2. External and internal style sheets (in the head section)
  3. Browser default

As a result, an inline style takes precedence over external and internal styles, as well as browser defaults.
