HTML Introduction

What is HTML?

  • HTML stands for Hyper Text Markup Language.
  • HTML is the most commonly used markup language for creating Web pages.
  • HTML consists of a several of elements.
  • HTML elements instruct browsers on how to display content.
  • The labels "this is a heading," "this is a paragraph," "this is a link," etc., are provided by HTML elements.

Simple HTML Document

<!DOCTYPE html>
<html>
<head>
<title> Page Title </title>
</head>
<body>

<h1>My first heading</h1>
<p>My first paragraph.</p>

</body>
</html>
Example Explained
  • <!DOCTYPE html> This document's declaration identifies it as an HTML5 document.
  • <html> element is the root element of an HTML Page
  • <head> element contains meta information about the HTML page
  • <title> element specifies a title for the HTML page
  • <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • <h1> element defines a large heading
  • <p> element defines a paragraph

HTML Element

Start tag Element content End tag
<h1> My first heading </h1>

Note: Some HTML elements, such as the <br> element, have no content. These are known as empty elements. There is no end tag for empty elements!
