CSS Syntax

A CSS rule is made up of two parts: a selector and a declaration block.

Syntax



The selector indicates the HTML element to be styled.

One or more declarations are separated by semicolons in the declaration block.

Each declaration contains a colon-separated list of CSS property names and values.

Semicolons are used to separate multiple CSS declarations, and curly braces are used to surround declaration blocks.

Example

In this example, all <p> elements will be center-aligned and have red text:

p {
    color: red;
    text-align: center;
}
Explanation of an Example
  • In CSS, p is a selector (it points to the HTML element to be styled: <p>).
  • color is a property, and red is the property value
  • text-align is a property, and center is the property value
